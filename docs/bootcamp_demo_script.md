# BootCamp Demo Script

## 目的

自然派やさいマップが、地域で自然派やさいを買える場所を入口に、農家・販売店・たね・地域のつながりを見える化するWebデモであることを短時間で伝える。

このデモでは、実装済みの検索体験と、将来構想としてのAI活用・地域やさいポイントを分けて説明する。

## デモで見せる順番

1. トップページ
2. 地域の販売店
3. 農家と出会う
4. 農家詳細
5. たねを見る
6. 農家さんへ
7. 販売店・飲食店向け
8. マイページ
9. この活動について

## 各ページで話す一言

### 1. トップページ

「一般ユーザーには、まず地域で買える野菜と販売店を探す入口として見せます。農家やたねは、興味が深まった人のための導線です。」

### 2. 地域の販売店

「自然食品店、直売所、飲食店、カフェなど、地域の野菜と出会える場所を整理するページです。今はサンプル表示で、実在店舗の詳細は本人確認後に掲載する想定です。」

### 3. 農家と出会う

「農家の考え方や育てているもの、つながり方を紹介します。単なる店舗検索ではなく、地域の農家と出会う入口にしています。」

### 4. 農家詳細

「公開情報をもとにした情報と、匿名モデル農家を分けて表示します。本人確認前のラベルを明確にして、断定しすぎないようにしています。」

### 5. たねを見る

「茨城に伝わる在来種・固定種・地域品種を、出典と確認状態を分けて表示します。位置は地域の目安で、正確な圃場所在地ではありません。」

### 6. 農家さんへ

「SNSやホームページを頻繁に更新していない農家でも、URLや写真、想いからプロフィール下書きを作れる構想です。AIは評価ではなく、本人確認前の下書き補助です。」

### 7. 販売店・飲食店向け

「農家の背景を、売場POPやメニュー説明の素材として伝えやすくする将来構想です。問い合わせ送信やAI生成はまだ実装していません。」

### 8. マイページ

「地域で買う、食べる、イベントに参加する行動を、将来的に地域やさいポイントとして扱う構想です。実際のポイント付与、QR読み取り、ログイン、決済連携はまだありません。」

### 9. この活動について

「一般ユーザー向けの入口の裏側に、農家・販売店・たね・地域資料を整理する運営構想があります。AIは運営負担を下げる補助として控えめに位置づけています。」

## 注意して説明すること

- AI機能はすべて将来実装・構想デモであり、実際のAI連携はまだありません。
- 地域やさいポイントは工事中で、実際のポイント付与や決済連携はまだありません。
- 店舗・農家・たね情報は、本人確認前、地域資料、公的DBなどの確認状態を分けて表示しています。
- 「無農薬」「安心安全」などの断定表現は避け、必要な場合は誤解の少ない表現に整える方針です。
- 農家を評価・採点するサービスではなく、農家本人が確認・修正して掲載するための下書き補助です。

## フィードバックしてほしい観点

- 一般ユーザーに入りやすいか
- マニアックすぎないか
- 競合との差別化が伝わるか
- AI機能が「将来構想」として自然か
- 農家や販売店に嫌がられない表現か
- 自治体・地域団体に見せられるか
- トップページから販売店、農家、たねへの導線が分かりやすいか
- 本人確認前・構想デモ・工事中のラベルが十分に伝わるか
