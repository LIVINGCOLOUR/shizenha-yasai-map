# 自然派やさいマップ - Claude Code作業ルール

## プロジェクト概要
- プロジェクト名：自然派やさいマップ
- 公開URL：https://shizenha-yasai-map.pages.dev/
- 技術構成：静的HTML/CSS/JavaScript、Cloudflare Pages、Cloudflare Pages Functions、OpenAI API連携

## 絶対に守るルール

### 文章について
- ユーザーが指定した本文は一字一句そのまま反映する
- 文章を勝手に整えたり言い換えたりしない
- HTML構造・カード・余白・CSS調整は行ってよい

### Git操作について
- pushは絶対にしない
- commitまでは行ってよい
- 作業後は必ずgit diff / git statusを確認してからcommitする

### 作業完了前に必ず実施
- python -m py_compile tools/qa_check.py
- python tools/qa_check.py
- ローカル静的サーバー起動（python -m http.server 8000）
- ブラウザ確認
- Consoleエラー確認
- スマホ幅確認
- git diff / git status確認
- commitまで行う

### 作業後の報告項目
- 変更したファイル
- 新規作成したファイル
- 実装した内容
- 削除・変更した文言
- 導線変更
- QA結果
- ローカル確認結果
- ユーザーが確認すべきURL
- コミットID

## 表現方針

### 使わない言葉
- 無農薬
- オーガニック
- 安心安全
- 完全無農薬
- AI（表面上のページでは使わない）

### 使う言葉
- 自然に寄り添う
- 地域の野菜や果物
- 農家の背景
- 作り手の想い
- 下書き作成支援
- システムが整理する

## 農法・掲載方針
- 農法の優劣を決めない
- 有機JAS認証の有無だけで農家を評価しない
- 栽培方法を認証・保証するサイトではない
- 情報の誠実さを重視する

## 名前変更方針
- 表示名「やまだ農園」→「笠間農園」に変更済み
- id=yamada-noenなどのURLパラメータ・内部IDは変更しない

## アプリの方向性
- 検索アプリではなく体験アプリ
- 一般消費者が自分から検索するアプリではない
- レストラン・カフェ・道の駅・給食などの食体験を起点にする
- 農家・野菜・果物・買える場所・感謝・継続購入へつながる体験を目指す
