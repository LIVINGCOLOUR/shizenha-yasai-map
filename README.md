# 自然派やさいマップ

自然に寄り添う農家と、地域の食をつなぐWebデモです。

自然派やさいマップは、地域の野菜を「食べる・買う・知る」入口からたどり、
農家・飲食店・販売店・地域に根差した野菜・自治体をゆるやかにつなぎます。

・表向きは、一般消費者が地域の野菜を食べられる店や買える場所を探す地図です。  
・制作者の想いは、オーガニック系の自然に向き合う農家を応援し、地産地消や食料安全保障の機運向上を目指しています。

## デモで見られるもの

- トップページの3つの入口
  - 食べる
  - 買う
  - 農を知る
- 地域の販売店・飲食店ページ
- 農家一覧・農家詳細ページ
- 地域に根差した野菜を見るページ
- 茨城県の在来種・固定種・地域品種の概略マップ
- 農家さん向けプロフィール下書き生成デモ
- 販売店・飲食店向け機能の構想
- 自治体・地域団体向け活用の構想
- 地域やさいポイントの将来構想

## 農家さん向けプロフィール下書き生成

農家さんが短く話す・書く・URLを渡すだけで、掲載前のプロフィール下書きを作る機能を想定しています。

現在のデモでは、会話形式の入力画面から回答を集め、Cloudflare Pages Functions 経由で OpenAI API を呼び出し、農家プロフィールの下書きを生成します。

生成された内容は、そのまま公開するものではありません。  
農家さん本人が確認・修正したうえで掲載する前提です。

この機能では、次のような情報を整理します。

- 農園紹介
- 育てているもの
- 大切にしていること
- 栽培方法について
- 地域との関わり
- 販売店・飲食店向けの紹介素材
- 価値観タグ候補
- 表現チェックメモ
- 掲載前チェックリスト

## 表現方針

このデモでは、「オーガニック」「無農薬」などの言葉を前面には出していません。
やわらかく自然な雰囲気の表現にし、各農家のプロフィールや詳細情報の中で、
栽培方針が分かるようにする設計を想定しています。

たとえば、実運用では次のような情報を農家ごとに確認して掲載する想定です。

- 化学合成農薬を使っているか
- 化学肥料を使っているか
- 自然由来の防除資材を使っているか
- 除草剤を使っているか
- 在来種・固定種・地域品種を扱っているか
- 農家本人による確認済みか

## AI機能について

現在実装済みのAI機能は、農家さん向けのプロフィール下書き生成です。

一方で、以下はまだ構想段階です。

- 音声入力
- フォーム送信
- DB保存
- 実登録処理
- QR読み取り
- ポイント付与
- 決済連携
- 販売店・飲食店向け紹介文生成
- 地域農業レーダー
- 掲載候補ボード

APIキーはブラウザ側には置かず、Cloudflare Pages Functions 側で環境変数として扱う構成です。

## 注意

掲載している農家・販売店・地域に根差した野菜の情報には、デモ用・本人確認前・出典確認前の情報が含まれます。

実運用前には、農家本人・店舗・出典元の確認が必要です。

また、栽培方法に関する表現は慎重に扱う必要があります。  
「有機」「無農薬」「安心安全」などの表現は、確認できた範囲に基づき、断定しすぎない形で扱う想定です。
